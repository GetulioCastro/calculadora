# calculadora
Criando uma calculadora para realizar apenas as quatro operações: somar, subtrair, multiplicar e dividir;

Usaremos somente a biblioteca tkinter de interface gráfica para treinar minhas habilidades;

Ao longo do projeto, iremos implementar os testes, criação de classes e demais funções para ir incrementando esta calculadora e, quem sabe, implementar também, através de outras bibliotecas, as funcionalidades científicas de uma calculadora completa.
